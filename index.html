⚡ Firebase Setup Required for Real-Time Sync

This dashboard works fully in demo mode right now. To enable real-time collaboration and embed functionality across devices, connect a free Firebase project (takes ~3 minutes):

Go to console.firebase.google.com → Create project → Add a Web app
Enable Realtime Database and Authentication → Email/Password
Copy your config object and paste into the FIREBASE_CONFIG variable at the bottom of this HTML file
Set database rules so public reads are allowed and writes require signed-in users, then re-upload to GitHub Pages
Open Firebase Console
